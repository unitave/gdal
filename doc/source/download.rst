개발 소스
------------------------------------------------------------------------------

GDAL 주 저장소는 깃허브 https://github.com/OSGeo/GDAL 에 있습니다.

다음 명령어를 실행하면 현재 개발 중인 소스 코드의 복사본을 얻을 수 있습니다.

::

    git clone https://github.com/OSGeo/GDAL.git


빌드 요구 사항
..................

GDAL 3 이상 버전을 빌드하려면, *최소한* C++11 호환 컴파일러와 `PROJ 6 이후 버전 <https://proj.org>`_ 이 필요합니다.
이렇게 하면 다수의 드라이버가 없는 최소 빌드만 얻게 될 것입니다.
선택적인 의존성을 추가하려면 :ref:`raster_drivers` 및 :ref:`vector_drivers` 페이지를 참고하십시오.


빌드 지침
..................

기존 autoconf 및 nmake 빌드 체계에 대한 힌트를 바란다면 https://trac.osgeo.org/gdal/wiki/BuildHints 를 읽어보십시오.

GDAL 3.5 버전부터, *실험적인* :ref:`CMake 기반 빌드 <build_hints>` 를 이용할 수 있습니다.


바이너리
------------------------------------------------------------------------------

이 단락에서는 GDAL의 여러 바이너리 배포판들을 설명합니다.
모든 바이너리는 완전하게 재생산할 수 있는 오픈 소스 빌드 레시피를 가지고 있습니다.

윈도우
................................................................................

`Conda Forge`_ 에서 윈도우 (64비트 전용) 빌드를 다운로드할 수 있습니다.
더 자세한 정보는 :ref:`conda` 단락을 참조하세요.

.. _`Conda Forge`: https://anaconda.org/conda-forge/gdal

데비안
................................................................................

이제 `Debian Unstable`_ 에서 데비안 패키지를 다운로드할 수 있습니다.

.. _`Debian Unstable`: https://tracker.debian.org/pkg/gdal

.. _conda:

콘다(Conda)
................................................................................

다중 플랫폼(윈도우, 맥OS, 리눅스)에서 소프트웨어 패키지를 설치하고 소프트웨어 환경을 관리하는 `콘다 <https://anaconda.org>`__ 를 사용할 수 있습니다. https://anaconda.org/conda-forge/gdal 에서 GDAL 용 콘다 패키지를 다운로드할 수 있습니다.

.. only:: html

    최신 버전: |Conda badge|

    .. |Conda badge| image:: https://anaconda.org/conda-forge/gdal/badges/version.svg
        :target: https://anaconda.org/conda-forge/gdal

::

    conda install [-c channel] [package...]


::

    conda install -c conda-forge gdal


리눅스 도커 이미지
................................................................................

`도커 허브 <https://hub.docker.com/r/osgeo/gdal/tags>`_ 에서 GDAL 마스터의 나이틀리 빌드(nightly build)와 태그된 배포판 이미지를 다운받을 수 있습니다.

서로 다른 환경설정의 내용에 대한 정보를 알고 싶다면 `https://github.com/OSGeo/gdal/tree/master/docker <https://github.com/OSGeo/gdal/tree/master/docker>`_ 페이지를 방문하십시오.
