================================================================================
LOSLAS -- NADCON .los/.las 원점 그리드 이동(Datum Grid Shift)
================================================================================

.. shortname:: LOSLAS

.. built_in_by_default::

GDAL 3.1버전부터, https://geodesy.noaa.gov/GEOID/MEXICO97/ 같은 .geo 확장자를 가진 지오이드 모델도 열 수 있습니다.

주의: ``gdal/frmts/raw/loaslasdataset.cpp`` 로 구현되었습니다.
