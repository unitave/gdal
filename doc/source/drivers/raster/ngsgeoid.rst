================================================================================
NGSGEOID - NOAA NGS 지오이드 표고 그리드
================================================================================

.. shortname:: NGSGEOID

.. built_in_by_default::

GDAL은 NOAA NGS 지오이드 표고 그리드(Geoid Height Grids) 바이너리 포맷 (.bin) 읽기를 지원합니다. 이 포맷은 수직 원점(datum) 변환에 쓰일 수 있습니다.

드라이버 케이퍼빌리티
-------------------

.. supports_georeferencing::

.. supports_virtualio::
